# Proyecto PHP

## Descripción

Este es un programa para un sitio web con un saludo y una imagen.

## Prerrequisitos

Instalar apache2
Codigó de instalación es sudo apt install apache2

## Pasos para su ejecución

1. Poner el codigo en el compilador.
2. En el explorador poner http://localhost/ProyectoPHP/index.php
3. El sitio web se desplegara automaticamente.
